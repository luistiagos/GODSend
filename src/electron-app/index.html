Settings
        

        


          

            
          


          

            Xbox connection
            
            
            
            
            
            
            
            

              
              
            

            

              
            

            

            
Enable FTP in Aurora (Settings → Network → Enable FTP) before using this button. Your PC's IP is detected automatically and patched into GODSend.ini. The path must match the folder Aurora actually loads (copy it from your FTP client). On USB that is often /Usb0/Apps/Aurora/User/Scripts/Utility/GODSend — note Apps and Utility (not Utilities). On HDD it is often /Hdd1/Aurora/User/Scripts/Utility/GODSend. Credentials and destination path are saved when you click the button above.

          


          

            Local Transfer folder (ISOs)
            

              
              
              
            

            
Changing this restarts the backend. The Xbox script uses this folder for "Local Library".

          


          

            Game cache
            

            

              
            

            
Caches are loaded from disk on startup and never refreshed automatically. Click to re-fetch all Internet Archive game lists and any ROM system caches you have previously browsed.

          


          

            Internet Archive account
            

            
            
            
            
            

              
              
            

            
Uses archive.org's official login API. Session cookies are saved locally; your password is never stored.

          


          

            ROM install path (on Xbox)
            

              
              
              
            

            
Drive-relative path for ROM installs. Each system gets a subfolder (e.g. \NES\, \SNES\). Changing this restarts the backend.

          


          

            
            
            
Simultaneous range-request workers per IA download (1–7). Higher is faster but puts more load on archive.org. Restarts the backend when changed.
